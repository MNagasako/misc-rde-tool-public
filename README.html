📦 同梱ソフトウェアとライセンス

本アプリケーションは、Python および各種第三者ライブラリを利用して構築されています。

📁 ライセンスファイルの格納場所

本アプリケーションに含まれる第三者ライブラリのライセンス情報は、以下のフォルダに格納されています：

LICENSES/

  • THIRD_PARTY_LICENSES_SUMMARY.txt - パッケージ一覧とライセンス種別のサマリ
  • README_LICENSES.txt - ライセンス情報の検出ロジックと注意事項
  • 各パッケージのライセンス全文（パッケージ名-バージョン-LICENSE.txt形式）

🔍 ライセンス情報の確認方法

  1. LICENSES/THIRD_PARTY_LICENSES_SUMMARY.txt を開いて、利用しているパッケージとライセンス種別を確認してください。
  2. 詳細なライセンス全文は、LICENSES/ フォルダ内の各ファイルを参照してください。
  3. ライセンス情報の検出方法や注意事項は、LICENSES/README_LICENSES.txt をご覧ください。

📋 主要な第三者パッケージ

本アプリケーションは、以下のような第三者Pythonパッケージを利用しています：

  • PySide6 (LGPL-3.0-only 他) - Qt for Python GUIフレームワーク
  • requests (Apache-2.0) - HTTPライブラリ
  • pandas (BSD 3-Clause) - データ解析ライブラリ
  • numpy - 数値計算ライブラリ
  • openpyxl (MIT) - Excelファイル操作
  • pillow (MIT-CMU License) - 画像処理ライブラリ
  • PyYAML (MIT) - YAML解析
  • truststore (MIT) - SSL証明書管理
  • その他多数のサポートライブラリ
