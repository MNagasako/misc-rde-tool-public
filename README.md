## 主な機能
- **データ取得**: RDEから登録済みデータセットの書誌情報JSONと画像サムネイル取得・JSON匿名化
- **基本情報取得・XLSX出力**: アクセス可能な全データセット情報の取得・Excel形式での書き出し
- **データ登録**: ファイルアップロード機能（uploadId重複対策済み）
- **サブグループ作成**: 研究グループの管理・メンバー権限設定
- **データセット開設**: エンバーゴ設定・広域シェア・匿名化オプション対応
- **AIテスト機能**: プロンプト入力・拡大表示・キーボードショートカット対応
- **設定・リクエスト解析**: 開発・デバッグ用機能

### 1. ログイン
<img width="1062" height="698" alt="2025-07-31_10h49_43" src="https://github.com/user-attachments/assets/c9e7f1c9-cb3f-4f39-9572-693378c33d40" />
RDEアカウントでログインしてください。自動ログインしたい場合は、`input/login.txt` ファイルの先頭に以下の内容を保存してください（DICEアカウントのみ対応）。

```txt
username(mail)
password
dice
```

---

### 2. データ取得
ARIM課題番号を入力し「実行」ボタンをクリックすると、該当課題のデータセットJSONや画像ファイルを取得できます。

- JSONファイルは匿名化版が自動生成されます
- 複数課題の一括処理に対応
- `input/login.txt` に課題番号を事前記載可能

---

### 3. 基本情報取得・XLSX出力
「基本情報」メニューでアクセス可能なデータセット情報を取得し、Excel形式で出力できます。

- **ALL**: ログイン中ユーザーがアクセス可能な全データセット
- **検索**: キーワード指定またはユーザー自身のデータセットのみ
- **XLSX出力**: 取得したデータセット情報をExcel形式で書き出し
- **進行確認**: `output/rde/data/datasets` と `output/rde/data/dataEntry` で進行状況確認可能

---

### 4. サブグループ作成
研究グループの作成・管理を行います。

- グループメンバーの権限設定（オーナー・代理管理者）
- グループ名・説明・課題番号の設定
- メンバー情報の事前登録機能（`login/rde-member.txt`）

### 5. データセット開設
新規データセットの作成を行います。

- サブグループ選択（補完検索対応）
- エンバーゴ終了日・テンプレート設定
- 広域シェア・匿名化オプション
- 作成後の自動情報更新

### 6. データ登録
データセットへのファイル登録を行います。

- データ名・説明・実験ID等の詳細情報入力
- データファイル（必須）・添付ファイル（任意）の選択
- 同一データセットへの連続登録対応
- uploadId重複対策済み

### 7. AI機能・その他
- **AIテスト**: プロンプト入力・拡大表示・ショートカット機能
- **設定**: 各種設定項目（開発中）
- **リクエスト解析**: REST API解析（開発用・要パスワード）
