# ARIM RDE Tool - Release Notes v2.4.1

**リリース日**: 2026年1月2日

---

## 概要

v2.4.1 は、データ取得2の「メール通知」タブを運用しやすく改善するリリースです。
抽出範囲のプリセット、テスト/本番の運用モード明確化、通知先選択、送信ログ閲覧により、
誤送信防止と監査性を高め、実運用の手間を下げることを目的としています。

---

## 変更点（概要）

### ✉️ データ取得2: メール通知タブ改善

- 抽出条件の「範囲（日）」を初期値7日・最大999日に変更し、プリセット（1/3/7/30/180/365日）で素早く切替可能に。
- 運用モードを「テスト/本番」のラジオ選択に変更し、誤送信防止と運用意図を明確化。
- 本番送信先を「投入者/所有者/両方」から選択可能に。
- 送信ログの閲覧機能を追加（重複送信防止は「送信先+件名」キーで維持）。

### ⚙️ メール設定: From表示名

- Gmail/SMTP の From表示名（任意）を設定可能にし、メールヘッダへ反映。
