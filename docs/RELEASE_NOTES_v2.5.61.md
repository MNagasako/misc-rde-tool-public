# RELEASE NOTES v2.5.61

v2.5.61 は、データポータル書誌情報 JSON の匿名化品質を高め、匿名化済みファイル出力の安定性を向上させるリリースです。

## 変更点

### データポータル書誌情報 JSON 匿名化の改善
- `included` 内の `type == user` 要素を、属性マスクではなく要素ごと除去するよう改善。
  - 利用者情報が残存しない形で匿名化できるように調整しました。
- user 要素除去のみが差分だった場合でも、匿名化済み JSON を出力するよう改善。
  - 非開示ファイルの生成漏れを抑制し、アップロード前確認の安定性を向上しました。

### 品質面
- 匿名化器の単体テストと、データセットアップロード経路の関連テストを追加。
  - 既存機能への影響を抑えつつ、匿名化処理の回帰を検出しやすくしました。

## 動作確認環境

- Windows 10/11 64bit
- Python 3.13.5（ソース実行時）
- PyInstaller ワンファイルバイナリ
