# RELEASE NOTES v2.0.0

リリース日: 2025-11-07

主な変更点 (破壊的変更あり)

- PyQt5 → PySide6 へ完全移行（API/挙動に一部差分があります）
- ライセンスを GPLv3 から LGPLv3 に変更
- 配布方針の変更：公開リポジトリはバイナリ配布を主目的とし、今後はソースコードの公開を行いません（開発リポジトリは別途管理）
- データポータル統合機能の継承（v1.20.0 の機能を基に安定化）

移行に伴う注意点

- PySide6 に移行したため、カスタム UI 振る舞いや WebView 周りの実装に細かな差異が発生する可能性があります。
- 開発者向けビルド手順は開発リポジトリ側で管理されます。配布リポジトリではバイナリ（exe）を提供します。
- 旧バージョン向けのドキュメント（v1.x 系）は `docs/archive/` に退避しています。アプリの基本的な使用方法は旧版の説明とほぼ同じですが、変更点は上記を参照してください。

配布物

- arim_rde_tool v2.0.0 — バイナリは開発リポジトリから後日配布予定（本配布リポジトリには v1.20.0 のバイナリを暫定で保持）

問い合わせ

問題や不整合を発見した場合はリリースノートと README を参照の上、配布元へご連絡ください。
