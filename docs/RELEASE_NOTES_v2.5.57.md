# RELEASE NOTES v2.5.57

v2.5.57 は、Basic Info の samples 取得導線を強化し、取得経路の可観測性と診断性を改善したリリースです。

## 変更点

- Basic Info の samples 取得で、既存互換を維持したまま直接取得ルートを追加。
- ALL 実行とサンプル単体実行の双方で、新旧ルートを順次実行する導線に統一。
- 進捗表示に取得経路の識別情報を追加し、処理状況を追跡しやすく改善。
- Material token 未取得時のメッセージを改善し、原因切り分けをしやすく整理。

## テスト

- 関連ユニットテストを更新し、samples 直接取得ルートと導線統合の回帰を確認。
