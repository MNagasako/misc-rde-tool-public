# RELEASE NOTES v2.0.6

リリース日: 2025-11-13

主な変更点

- print()からlogger統一化完了（全ファイル対応）
- ロギング品質向上・デバッグ性改善
- v2.0.5継承機能の安定性維持（truststore統合SSL検証・APIログ機能・プロキシ再起動プロンプト）
- 既存機能の堅牢性向上・継続的品質改善
- バージョン管理の統一性維持

技術的詳細

- **ロギング統一化**: 全ソースファイルでprint()をloggerに置き換え、統一的なログレベル管理を実現
- **デバッグ性向上**: DEBUG/INFO/WARNING/ERRORの適切な使い分けにより、トラブルシューティングが容易に
- **品質改善**: v2.0.5で実装されたtruststore・APIログ・プロキシ管理機能の安定性を維持しつつ、コード品質を向上
- **継承機能**: TokenManager（v2.0.1）、truststore統合（v2.0.5）、データポータル統合（v1.20.0）の全機能を維持

移行・利用上の注意

- v2.0.5以前の操作方法・基本機能はそのまま利用可能です。
- ログ出力がより詳細になり、デバッグが容易になりました。
- 詳細な技術仕様や開発情報は開発リポジトリを参照してください。
