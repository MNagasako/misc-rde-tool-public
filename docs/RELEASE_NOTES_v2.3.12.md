# ARIM RDE Tool - Release Notes v2.3.12

**リリース日**: 2025年12月29日

---

## 概要

v2.3.12 は、AI説明文提案の「結果一覧」表示を改善し、クリックでログ内容（JSON/テキスト）を確認できるようにしたリリースです。
また、対象=報告書における欠損値（nan）の表示を空値へ正規化し、挙動をテストで固定しました。

---

## 変更点（概要）

### 🤖 AI説明文提案（結果一覧）

- 結果一覧テーブルの行クリックで、対応ログ内容（JSON/テキスト）を表示。
- 対象=報告書の「重要技術領域（主/副）」が欠損（nan）の場合、空値表示へ正規化。

### 🧪 テスト/品質

- ログ表示（JSON/テキスト）および nan→空値 の挙動をウィジェットテストで固定。
