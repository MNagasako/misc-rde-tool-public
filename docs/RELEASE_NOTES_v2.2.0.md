# リリースノート v2.2.0

**リリース日**: 2025年12月6日

---

## 概要
v2.2.0 は、基本情報タブの処理完全性・可視化の強化、AI機能の安定化、データセット管理の操作性向上、設定・ネットワーク周りの堅牢化をまとめた包括的改善リリースです。

---

## 主な改善
### 🚀 基本情報タブの並列化・完全性強化
- サブグループ完全性チェックを強化し、欠損検知時の再取得ループを最適化。
- 進捗表示を細分化して処理ステップごとに可視化し、大規模プロジェクトでも把握しやすく改善。
- 並列化を最適化し、取得と再取得の両方で安定した処理を実現。

### 🤖 AI機能安定化
- `llm_model_name` プレースホルダ置換を完全対応し、設定変更時の置き換え漏れを解消。
- AI CHECK 系処理のタイムアウト・リトライ・エラー表示を調整し、結果の安定性を向上。

### 📑 データセット管理の改善
- 関連データ/関連情報ビルダーダイアログの入力補助とバリデーションを強化。
- データセット・データエントリーの日時表示を日本時間（JST）に統一し可読性を向上。
- Completer フィルタ選択や2段階検索の最適化を継承し、検索性を改善。

### ⚙️ 設定・ネットワーク・CLI の強化
- 設定タブ構成の整理と暗号化保存を維持し、環境依存性を低減。
- CLI オプション（ログレベル指定など）とヘルプ表示を整備。
- truststore 完全対応を維持し、企業プロキシ・証明書環境での安定性を確保。

---

- 開発リポジトリにて 458 件のユニットテストを実行し全合格（widget テストを含む）。

## 互換性・注意点

- グループ選択ダイアログはUIスレッド実行前提です。スクリプトから利用する場合もUIコンテキストで呼び出してください。
