# ARIM RDE Tool - Release Notes v2.4.10

**リリース日**: 2026年1月7日

---

## 概要

v2.4.10 は、データセット一覧タブの操作性とパフォーマンス改善、およびエクスポート UX 改善を含むリリースです。

---

## 変更点（概要）

### 🧩 データセット一覧タブ: フィルタUI改善

- 範囲指定3項目を1行にまとめ、入力欄間の表記を「～」に統一。
- フィルタ領域の折りたたみに対応し、折りたたみ時は入力済み条件のみ要約表示。

### ⚡ データセット一覧タブ: 表示高速化

- 一覧生成に signature ベースのインメモリキャッシュを導入し、再表示を高速化。

### 📦 エクスポートUX

- CSV/XLSX エクスポート時に、日時付きの既定ファイル名を保存ダイアログへサジェスト。

---

## `latest.json` について

- `latest.json` は GitHub Releases の配布物（インストーラ exe + sha256）と連動します。
