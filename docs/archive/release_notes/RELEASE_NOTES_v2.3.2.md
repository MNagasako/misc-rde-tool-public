# ARIM RDE Tool - Release Notes v2.3.2

**リリース日**: 2025年12月21日

---

## 概要

v2.3.2 は、説明欄などのテキストエリア（QTextEdit/QTextBrowser/QPlainTextEdit）で、
枠線が見えない・背景が周囲と同化して区別しづらい、といった視認性問題を改善するアップデートです。
環境差の影響を受けにくい適用経路へ寄せ、ウィジェットテストで回帰を防止します。

---

## 主要な改善点

### ✍️ テキストエリア可視性（枠線/背景）

- **枠線表示の安定化**: `::viewport` セレクタが環境により効きにくいケースを吸収するため、viewport へスタイルを直接適用するフォールバックを追加。
- **ライトテーマでの背景同化を解消**: テキストエリア背景を周囲より明確に明るい色へ調整し、入力欄としてのコントラストを確保。

### 🧪 テスト・品質管理

- 枠線（border）と背景（background）の描画結果を `grab()` のピクセル検証で確認するウィジェットテストを追加。
- 「スタイル文字列にborderがある」ではなく「実際に描画される」ことをテストで保証。

---

## テスト状況

バージョンアップ時は、少なくとも以下の確認を推奨します。

- `./.venv/Scripts/python.exe -m pytest -q -k 'test_revision_sync'`
- `./.venv/Scripts/python.exe -m pytest -q`

---

## 既知の制限

- Windows環境での Qt イベント処理に関する稀なクラッシュ（0x8001010d）は、テスト実行ログに出る場合があります（継続的に対応中）。
