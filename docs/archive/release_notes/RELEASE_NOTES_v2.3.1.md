# ARIM RDE Tool - Release Notes v2.3.1

**リリース日**: 2025年12月20日

---

## 概要

v2.3.1 は、データセット編集機能の改善アップデートです。データセット更新後の再取得、キーボード操作対応、コンボボックスのアイテム管理修正により、利便性を向上させました。

---

## 主要な改善点

### 📊 データセット機能改善

- **API再取得による最新データ表示**: データセット更新後にRDE APIから最新データを再取得して表示するように改善（古いキャッシュ表示を低減）。
- **キーボード操作対応**: データセット編集コンボボックスでカーソルキー（↑↓）・PageUp/PageDown・マウスホイール操作に対応。
- **アイテム管理の修正**: コンボボックス本体に全アイテムを正しく格納し、件数プレースホルダー表示と操作性の不整合を解消。

### 🧪 テスト・品質管理

- API再取得周りのテスト（ユニットテスト）追加と、monkeypatch戦略の見直しによりテストのimportエラーを解消。

---
