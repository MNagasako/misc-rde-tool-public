# ARIM RDE Tool - Release Notes v2.3.9

**リリース日**: 2025年12月28日

---

## 概要

v2.3.9 は、Windows/Qt 環境での長時間テストスイート実行時に発生しうる不安定要因を抑制し、テーマ追従（theme_changed）周りの破棄後コールバックを防ぐことで、フルスイートの完走性を改善したリリースです。あわせてドキュメント/配布物のバージョン表記を更新しました。

---

## 変更点（概要）

### 🧪 テスト安定化（Windows/Qt・長時間スイート）

- 長時間のpytest実行で発生しうる「破棄済みQtオブジェクト参照」起因の不安定化を抑制（destroyed時のdisconnect/弱参照化/生存チェック）。
- Windowsのウィジェットテストがネイティブウィンドウ資源に影響されにくいよう調整し、フレークを低減。

### 🎨 UI/テーマ

- テーマ変更シグナルの接続管理を整理し、破棄後コールバックによるエラー/不安定化を抑制（通常実行の挙動は維持）。

---

## テスト状況

- 全テストスイートを実行し合格（環境依存の skip は継続）。
