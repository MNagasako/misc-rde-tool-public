# ARIM RDE Tool - Release Notes v2.3.15

**リリース日**: 2025年12月31日

---

## 概要

v2.3.15 は、AI説明文提案機能の操作性向上（データセット向けタブ追加）と、結果一覧から参照するログビューアの改善を含むリリースです。
加えて、pytest 実行時のテーマ切替に伴う負荷を抑制し、長時間テストの安定性を改善しました。

---

## 変更点（概要）

### 🤖 AI説明文提案: データセットタブ追加

- AI説明文提案ダイアログに「データセット」タブを追加。
- 従来のコンボ選択方式から、「一覧テーブル選択 + 列別フィルタ」方式へ移行。
- 一括問い合わせ/並列問い合わせ（キャンセル含む）の操作フローに対応。

### 🧾 ログビューア改善（結果一覧）

- 結果一覧から開くログ表示ダイアログで、JSONログの表示を切替可能に改善。
  - ツリー表示（キー階層）
  - JSON全文表示
- 「標準テキストエディタで開く」ボタンを追加。

### 🧪 テスト/安定性（テーマ切替）

- pytest 実行時のテーマ切替で、重いグローバルQSS適用やトップレベル更新の影響を抑制。
- 長時間スイートでのタイムアウト発生を低減。
