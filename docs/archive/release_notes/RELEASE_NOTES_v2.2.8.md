# ARIM RDE Tool - Release Notes v2.2.8

**リリース日**: 2025年12月18日

---

## 概要

v2.2.8 は v2.2.7 に対する追加改善リリースです。
データセット開設（新規開設/新規開設2）のフィルタUI改善と、基本情報キャッシュ再利用時のサブグループ完全性判定の安定化を行いました。

---

## v2.2.7 からの差分（主な変更点）

### 🧩 データセット開設フォーム（新規開設/新規開設2）UI

- ロール/サブグループ/テンプレート/組織フィルタのラベル文言を整理
- ロールフィルタに「フィルタなし」を追加し、絞り込み解除を明示

### 📊 基本情報: キャッシュ判定の安定化

- サブグループ0件ケースで subGroups/ を欠損扱いにせず、不要な再取得を抑制
  - 既存キャッシュがあるのに「subGroups/配下にファイルが無い」だけで pipeline を再実行してしまうケースを改善

### 📚 ドキュメント更新

- README/CHANGELOG/リリースノートを v2.2.8 に更新
- v2.2.8 のリリースノート（本ファイル）を追加

---
