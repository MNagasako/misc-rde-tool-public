# RELEASE NOTES v2.4.12

v2.4.12 は、一覧タブの大規模データ環境で発生し得る「無限スピナー（固まって見える）」問題の抑止と、非同期リロードの安定化を含むリリースです。

## 変更点

- 一覧タブ（サブグループ一覧 / データセット一覧）
  - 大規模データで重くなりがちな処理を抑制し、初回表示が終わらないように見えるケースを低減。

- 安定性
  - 非同期リロードのスレッド寿命管理を改善し、タブ切替等での `QThread: Destroyed while thread is still running` 発生を抑止。

## テスト

- ユニット/ウィジェット/統合テストを実行し、回帰を防止。
