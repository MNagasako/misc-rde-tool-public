# ARIM RDE Tool - Release Notes v2.3.10

**リリース日**: 2025年12月28日

---

## 概要

v2.3.10 は、AI説明文提案（対象=報告書）の「結果一覧」表示を改善し、重要技術領域（主/副）列を追加したリリースです。
あわせて、関連テストを実データ/キャッシュに依存しない形へ調整し、フルスイートでの決定性を強化しました。

---

## 変更点（概要）

### 🤖 AI説明文提案（報告書）

- 「結果一覧」タブで対象=報告書の場合、`target_key` の ARIM課題番号と報告書リストを結合し、
  「重要技術領域（主）」および「重要技術領域（副）」の列を追加表示。

### 🧪 テスト

- 報告書タブ/結果一覧タブのウィジェットテストがローカルの実データやキャッシュに影響されないよう決定性を改善。
