# RELEASE NOTES v2.4.17

v2.4.17 は、配布物（インストーラ）と更新情報（`latest.json` / SHA256）の更新を含むリリースです。

## 変更点

- 配布物
  - インストーラを更新し、`latest.json` と SHA256（`checksums.sha256`）を更新。

- テーマ（v2.4.16 相当の内容を含みます）
  - テーマ切替時に特定の破棄順序でクラッシュする問題を修正。
  - 一覧タブ/各種ウィジェットでローカルQSSがテーマ切替に追従しない問題を修正。
  - データポータル設定のステータス表示は、文字列ごとの固定色指定をやめ、テーマに追従する見た目に統一。

## テスト

- ユニット/ウィジェット/統合テストを実行し、回帰を防止。
