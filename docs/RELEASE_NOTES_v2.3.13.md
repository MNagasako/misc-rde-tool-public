# ARIM RDE Tool - Release Notes v2.3.13

**リリース日**: 2025年12月29日

---

## 概要

v2.3.13 は、設定→データ構造化（対応ファイル形式: XLSX）における「テンプレートID/名」の取得ロジックを改善し、
テンプレート列が V6/V7 のように混在している場合でも、行ごとに適切な列を参照できるようにしたリリースです。

---

## 変更点（概要）

### ⚙️ 設定: データ構造化（XLSX）

- テンプレートID/名の取得方法を改善。
  - 従来: 「最新列固定」（例: V7 列のみ参照）
  - 今回: 行単位で「値が入っている最大 Vn」の列を採用（V6/V7 混在に対応）

### 🧪 テスト/品質

- V6/V7 混在ケースのユニットテストを追加し、回帰を防止。
