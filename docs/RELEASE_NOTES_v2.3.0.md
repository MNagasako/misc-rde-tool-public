# ARIM RDE Tool - Release Notes v2.3.0

**リリース日**: 2025年12月20日

---

## 概要

v2.3.0 は、ログイン処理のさらなる安定化とテストスイートのオフライン完走性向上、ならびにバージョン表記の全体統一を行ったアップデートです。

---

## 主要な改善点

### 🔐 自動ログイン・安定化の継続改善

- WebView の storage へトークンが出現するまで待機（バックオフ + 総タイムアウト）する方式を継続強化。
- **自動ログイン時のみ**、RefreshToken がある場合のフォールバック挙動を最適化。

### 🧪 テストのオフライン安定化

- pytest 実行時の外部 HTTP 依存をより抑制し、オフライン環境でもテストスイートの完走性を向上。
- Windows 環境での 0x8001010d 系クラッシュ回避の調整を維持。

### 📚 バージョン表記の統一・ドキュメント更新

- VERSION.txt / README / CHANGELOG / リリースノート等の表記を v2.3.0 に統一。

---

## テスト状況

バージョンアップ時は、少なくとも以下の確認を推奨します。

- `./.venv/Scripts/python.exe -m pytest -q -k 'test_revision_sync'`
- `./.venv/Scripts/python.exe -m pytest -q`
