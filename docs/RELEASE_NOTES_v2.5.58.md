# RELEASE NOTES v2.5.58

v2.5.58 は、Basic Info のエクスポート導線と進捗表示の安定化を中心としたリリースです。

## 変更点

- エクスポート開始時の保存導線を標準化し、保存先確定後に進捗ダイアログを表示する流れへ統一。
- 保存先の既定値を Desktop とし、2回目以降は前回保存先を初期値として再利用。
- 進捗更新を UI スレッド側で反映する構成に整理し、表示の安定性を改善。
- 進捗ダイアログに経過時間と残り時間（推定）を表示。
- samples 新ルート実行条件を整理し、既定導線での挙動を明確化。

## テスト

- 関連ユニットテストおよびウィジェットテストを更新し、導線変更と進捗表示の回帰を確認。
