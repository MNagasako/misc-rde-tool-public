# ARIM RDE Tool - Release Notes v2.2.6

**リリース日**: 2025年12月13日

---

## 概要

v2.2.6 は、データポータルのデータセットアップロードUI改善と、データセット編集TAGビルダーへのAI提案タブ追加を中心とした改善リリースです。併せて、配布用ドキュメントのバージョン表記を 2.2.6 へ統一しました。

---

## 主要な改善点

### 🗂 データポータル: データセットアップロードUI改善

- 取得済み画像一覧を4列テーブル化（選択/ファイル名/キャプション/アップロード）
- キャプション編集、ヘッダ固定、ソート、アップ済み「Up済」表示に対応
- アップロード時にキャプションを反映（空の場合はファイル名を使用）

### 🏷 データセット編集: TAGビルダーAI提案タブ

- 指定プロンプトでAIへ問い合わせし、候補タグ（label）を一覧表示して採用できるように改善
- 「リトライ提案」「プロンプト全文」「回答全文」「全て採用」を追加
- 重複タグは自動的に除外

### 📚 バージョン表記統一

- VERSION.txt / README / CHANGELOG / リリースノート等の表記を v2.2.6 に統一

---

## テスト状況（推奨）

- `./.venv/Scripts/python.exe -m pytest -q -k 'test_revision_sync'`
- `./.venv/Scripts/python.exe -m pytest -q -k 'tag_builder_ai'`

---

## アップグレード手順

1. Releases ページから `arim_rde_tool_setup.exe`（v2.2.6）を取得し、インストーラーを実行します。
2. 既存環境で上書きインストールする場合は、インストール先の権限やセキュリティソフトのブロック状況に留意してください。
3. 起動後、データポータルのアップロード画面と、データセット編集のTAGビルダー（AI提案タブ）を確認します。
