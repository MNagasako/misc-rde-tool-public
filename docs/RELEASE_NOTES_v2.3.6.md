# ARIM RDE Tool - Release Notes v2.3.6

**リリース日**: 2025年12月24日

---

## 概要

v2.3.6 は、データ登録（通常登録/一括登録）のUXと完了表示の挙動を整理し、利用者の体感と要件に沿う形へ改善したリリースです。

---

## 変更点（概要）

### ✅ 通常登録

- 必須未入力でも押せる導線を維持しつつ、未入力の可視化（項目名列挙）と押下時アラートで入力を促す。
- 完了時は「確認」押下まで閉じない（他操作をブロックし、結果を確実に読める）。

### ✅ 一括登録

- 確認回数を減らすため、完了時は確認を要求せず自動クローズ寄り。

---

## テスト状況

バージョンアップ時は、少なくとも以下の確認を推奨します。

- `./.venv/Scripts/python.exe -m pytest -q -k 'test_revision_sync'`
- `./.venv/Scripts/python.exe -m pytest -q`
