# ARIM RDE Tool v2.1.11 - リリースノート

**リリース日**: 2025年11月30日

---

## 概要

v2.1.11 は、**設定タブの整理**、**ヘルプドキュメントの全面刷新**、**ウィンドウ位置ポリシーの確定**、および**コマンドライン引数の拡張**を含むマイナーアップデートです。より使いやすく、分かりやすいアプリケーションを目指した改善が行われています。

---

## 主な新機能・改善

### ⚙️ 設定タブの改善

#### 1. **設定タブ構成の整理**

設定画面が2つのタブに分離され、設定項目の視認性が向上しました。

**「基本設定」タブ**:
- テーマ選択（Dark/Light/System）
- ログレベル選択（DEBUG/INFO/WARNING/ERROR/CRITICAL）
- ウィンドウ位置設定（前回位置復元/センタリング/OS任せ）

**「ネットワーク設定」タブ**:
- プロキシ設定（AUTO/MANUAL/HTTP/OFF）
- SSL証明書設定（企業CA証明書パス）
- 接続テスト機能

#### 2. **ウィンドウ位置ポリシーの確定**

アプリケーション起動時のウィンドウ位置を3つのモードから選択可能になりました。

**動作モード**:
- **前回位置復元**: 最後に終了した位置を記憶して復元（デフォルト）
- **センタリング**: 常に画面中央に配置
- **OS任せ**: オペレーティングシステムに配置を委ねる

**特徴**:
- 設定の永続化対応
- マルチモニター環境での動作安定性向上
- 画面外配置の自動補正

### 📚 ヘルプドキュメントの全面改訂

アプリケーション内のヘルプドキュメントが大幅に刷新されました。

**改善内容**:
- 最新機能の反映（AI拡張機能、データポータル統合機能等）
- 構成の見直しと可読性の向上
- トラブルシューティング情報の充実
- 具体的な操作手順の追加

### 🛠️ コマンドライン引数の拡張

#### **--log-level オプション追加**

起動時にログレベルを指定できるようになりました。

**使用方法**:
```powershell
arim_rde_tool.exe --log-level DEBUG
```

**選択可能なレベル**:
- `DEBUG`: 詳細なデバッグ情報を出力
- `INFO`: 一般的な情報メッセージを出力（デフォルト）
- `WARNING`: 警告メッセージのみ出力
- `ERROR`: エラーメッセージのみ出力
- `CRITICAL`: 致命的なエラーのみ出力

**用途**:
- デバッグ作業の効率化
- トラブルシューティング時の詳細ログ取得
- 本番運用時のログ量削減

## v2.1.10からの継続改善

### 🔍 検索機能の強化（v2.1.10）

- **Completerフィルタ選択完全対応**
  - データセット編集時の自動補完にフィルタ選択を適用
  - 2段階検索実装（初回: 全件検索、2回目以降: 選択フィルタのみ）
  - 大量データセット環境での操作性向上

### 🔗 データポータル機能強化（v2.1.9）

- **RDE存在確認機能**
  - データポータルからRDEデータセットへの逆引き確認
  - 既存データの重複登録防止
  - ボタンの自動無効化で誤操作防止

## ダウンロード

以下のファイルをダウンロードしてください:

- **arim_rde_tool_setup.2.1.11.exe**: インストーラー版（推奨）

---

## 動作要件

- **OS**: Windows 10/11 (64bit)
- **メモリ**: 4GB以上推奨
- **ストレージ**: 500MB以上の空き容量
- **ネットワーク**: インターネット接続必須（RDE/データポータルへのアクセス）

---

## インストール方法

### インストーラー版（推奨）

1. `arim_rde_tool_setup.2.1.11.exe` をダウンロード
2. ダウンロードしたファイルを実行
3. ウィザードに従ってインストール
4. デスクトップのショートカットから起動

---

## アップグレード方法

### 既存ユーザー向け

1. 新しいバージョンのインストーラーを実行
2. 既存のインストールに上書きインストール
3. 設定ファイル（`config/`, `input/`）は自動的に保持されます

**注意**: ウィンドウ位置設定は初回起動時にデフォルト値（前回位置復元）が適用されます。必要に応じて設定タブで変更してください。
