# RELEASE NOTES v2.5.60

v2.5.60 は、データポータル編集画面の詳細確認機能強化とデータセットアップロード時の可視化改善を中心としたリリースです。

## 変更点

### データポータル編集画面の改善
- 保存確認ダイアログにペイロードプレビュー表示を追加。
  - 編集内容をJSON形式で確認し、送信前に詳細な検証が可能に。
- チェックボックス系配列フィールドの全解除時にクリア用の送信値（`['']`）へ正規化する挙動を強化。
  - 意図しない値の送信を防ぎ、データポータル側のバリデーション精度向上。

### データセットアップロード画面の改善
- JSONキャッシュの作成日時・更新日時を表示機能を追加。
  - キャッシュの鮮度確認が容易になり、再取得判断がしやすく改善。

## テスト

- 関連ユニットテスト（ポータル編集ダイアログ・チェックボックス操作・キャッシュ表示）により回帰確認完了。
- 静的解析（Pyright）による型チェック・コード品質検証も全て合格。

## 動作確認環境

- Windows 10/11 64bit
- Python 3.13.5（ソース実行時）
- PyInstaller ワンファイルバイナリ
