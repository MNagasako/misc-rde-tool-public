# RELEASE NOTES v2.5.63

v2.5.63 は、報告書変換の一括処理を再開しやすくし、途中中断後の回復性を高めることを中心としたリリースです。

## 変更点

### 報告書変換レジューム回復の改善
- 破損した `*.tmp.xlsx` や不整合な `convert_progress.json` を検知した際に、安全に破棄して先頭から再開するよう改善。
  - 中断後に残った一時ファイルや進捗情報が原因で、再開時に失敗し続けるケースを起こしにくくしました。
- チャンク保存を一時ファイル経由の原子的保存に変更。
  - 途中保存のタイミングでファイルが壊れるリスクを抑え、長時間処理の安定性を向上しました。

### 品質面
- 回帰テストを追加し、破損した一時 Excel からの回復と進捗メタデータ不整合時の再開位置補正を確認。
  - 報告書変換まわりの再開経路で、同種不具合を検出しやすくしました。

## 動作確認環境

- Windows 10/11 64bit
- Python 3.13.5（ソース実行時）
- PyInstaller ワンファイルバイナリ
