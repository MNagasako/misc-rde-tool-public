# ARIM RDE Tool - Release Notes v2.4.6

**リリース日**: 2026年1月6日

---

## 概要

v2.4.6 は、主に長時間実行（Windows + PySide6 + pytest-qt 環境）での安定性向上と、テスト運用手順の改善を含む内部改善リリースです。

---

## 変更点（概要）

### 🧪 テスト/品質（安定性）

- Windows/Qt 環境で断続的に発生していた不安定要因を減らすため、テスト待機・実行手順を見直し。
- フルスイートを「非widget → widget」に分割して実行する運用を追加し、単一コマンドでの完走性を改善。

### 📚 ドキュメント/表記

- `VERSION.txt` / `README.md` / `docs/CHANGELOG.md` / `docs/RELEASE_NOTES_v2.4.6.md` 等の表記を v2.4.6 に統一。

---

## 互換性

- 本リリースは機能追加というより内部改善が中心のため、ユーザー操作手順への影響は基本的にありません。
