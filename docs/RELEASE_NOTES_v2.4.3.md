# ARIM RDE Tool - Release Notes v2.4.3

**リリース日**: 2026年1月4日

---

## 概要

v2.4.3 は、メール通知タブの運用上の判別性と宛先運用を改善するリリースです。
運用モード表示を簡略化し、本番送信先に設備管理者を追加することで、
運用現場での誤解や通知漏れを減らすことを目的としています。

---

## 変更点（概要）

### ✉️ メール通知タブ（運用改善）

- 運用モード表示を「テスト/本番」に簡略化し、画面上の判別を明確化。
- 本番送信先に「設備管理者」を追加し、投入者/所有者に加えて宛先へ含められるよう改善。
