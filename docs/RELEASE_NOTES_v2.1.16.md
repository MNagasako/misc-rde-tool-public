# RELEASE NOTES v2.1.16 (2025-12-04)

## 概要

- バージョンリビジョン更新（2.1.16）
- データセット修正タブに「AI CHECK」ボタンを追加（説明文の品質チェック機能）
- AI CHECK 結果表示ダイアログのフォントサイズを改善（可読性向上）
- 公開配布ドキュメント更新（VERSION/README/CHANGELOG/本ファイル）
- 配布バイナリ: `dist/arim_rde_tool_setup.2.1.16.exe`

## 主な機能追加

### 🔍 説明文品質チェック機能（AI CHECK）

**新規ボタン**: データセット修正タブの説明文入力エリアに「AI CHECK」ボタンを追加

**チェック内容**:
- 説明文の妥当性を簡易評価（4段階評価）
  - ✅ 合格：十分に整備
  - 📝 微修正推奨：若干の改善を推奨
  - ⚠️ 要修正：修正が必要
  - ❓ 判定不能：条件不足等で判定不可

**技術実装**:
- AIRequestThread での非同期実行（UI ブロッキング回避）
- タイムアウト: 30秒（自動キャンセル）
- リトライ: 最大2回で信頼性向上
- JSON検証・修正・メッセージボックス表示

**エラーハンドリング**:
- 未選択状態の検出・警告
- 未入力チェック
- 解析エラー対応
- 実行エラー詳細表示

### 🎨 フォントサイズ改善（AI CHECK 結果ダイアログ）

**変更内容**:
| 要素 | 変更前 | 変更後 |
| --- | --- | --- |
| スコア表示 | 18px | 24px |
| 文字数表示 | 14px | 20px |
| 判定結果ラベル | 14px | 18px |
| テキストセクション | 10px | 11px |
| ボタン | 10px | 12px |
| 判定コメント表示領域 | 100px | 120px |

**その他改善**:
- テキストセクション: line-height 1.5 で行間を広げ読みやすく
- ボタン高さ: 36px に統一
- 詳細ダイアログラベル: 12px 太字で視認性向上
- テキスト: 11px で一貫性確保
