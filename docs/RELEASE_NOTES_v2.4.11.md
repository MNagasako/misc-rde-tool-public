# RELEASE NOTES v2.4.11

v2.4.11 は、サブグループ一覧タブの表示・操作性改善と、フィルタUIの安定化を含むリリースです。

## 変更点

- サブグループ一覧タブ
  - 「課題」「メンバー」「関連データセット」「関連試料」など、複数要素の表示を改行区切りに統一。
  - 関連データセット/試料をセル内の要素ごとに個別クリックで開けるよう改善。
  - 長い名称は省略表示 + ツールチップで全体表示（エクスポートは生値）。
  - 行高さが過大にならないよう、行内の最大行数に合わせて明示的に調整（画面高の50%で上限）。

- フィルタUI
  - 範囲フィルタの折り返し時にラベルが重なって見える問題を修正。

## テスト

- ユニット/ウィジェット/統合テストを実行し、回帰を防止。
